# Valentine's Card 💌

**Valentine's Card** is a fun and interactive web application that allows users to create personalized Valentine's greeting cards. Users can add their own messages, enjoy smooth animations, and share a heartfelt digital card with loved ones.

---

## Features ✨

- Create custom Valentine's messages.
- Beautiful animations for hearts and card effects.
- Responsive design – works on desktop and mobile.
- Lightweight and easy to use.

## Technologies Used 🛠️

- **HTML5** – structure of the card.
- **CSS3** – styling and animations.
- **JavaScript** – interactivity and dynamic message handling.
